Table2List | User Manual

This is a simple JavaScript library that converts HTML tables into lists for better responsiveness on smaller screens. Below are two example tables that will be converted into lists when the screen width is less than the specified breakpoints. Resize the window to see the effect.

 

Download the library from here.

Example Table 1

This is an example table that will be converted into a list when the screen width is less than 768px. Resize the window to see the effect.

let table = document.getElementById('mytbl');
new Table2List(table, 768);

ID Title Author Price Pages Language ISBN Category Published
1 Book 1 Author 1 10.99 100 English 1234567890 Category 1 2022-01-01
2 Book 2 Author 2 9.99 200 French 9876543210 Category 2 2022-02-02
3 Book 3 Author 3 8.99 300 Spanish 5678901234 Category 3 2022-03-03
