CP Player - Your MP3 Player Solution

This is a straight forward MP3 player build with HTML CSS and Javascript, It can be used on any website that uses audio podcast, audio tutorials and many more. It's very simple to setup. You can use it on your website by following simple steps.

Step 1: Add the following HTML code to your page: <div id="myplayer"></div>

Step 2: Download and Add the following JavaScript library to your page: <script src="cp-player.min.js"></script>

Step 3: Add the following JavaScript code to your page: // Create a playlist const playlist = [
'http://yourserver.com/assets/audio/1.mp3',
'http://yourserver.com/assets/audio/2.mp3',
'http://yourserver.com/assets/audio/3.mp3',
'http://yourserver.com/assets/audio/4.mp3',
];

// See all options for customization of player.
const options = {
useFontAwesome: true,
audioInfo: false,
...
};

// See all styleOptions for design customization.
const styleOptions1 = {
cpplayer: {
backgroundColor: '#118911',
border: 'none',
borderRadius: '0',
color: '#fff',
display: 'flex',
flexDirection: 'column',
alignItems: 'center',
justifyContent: 'center',
backgroundImage: "url('./assets/images/1.jpg')",
backgroundSize: 'cover',
backgroundPosition: 'center',
backgroundBlendMode: 'multiply',
width: '100%',
},
...
};

// Finally, Initialize Player with all the parameters above.
const player = new CP_Player(playlist, options, 'myplayer', styleOptions1);

Options


Option Description Type Default Value

showSoundControl Show / hide sound control Boolean false
seekControls Show / hide seek controls Boolean false
nextPreviousControls Show / hide next/previous controls Boolean false
playlist Show / hide playlist Boolean false
showTimeLapse Show / hide time lapse Boolean false
audioInfo Show / hide audio info Boolean false
layout change player layout String default / compact
